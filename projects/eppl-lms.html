CS394 · Biggest Project

EPPLMS

A learning management system built for a real school (Paragon International University). It handles students, lecturers, courses, classes, and a flexible system for deciding what each type of user is allowed to do.

by Yongye Uy · CS394 group project

Code is private (real school system)

what it doesOverview

The school needed its own system to manage students, lecturers, courses, semesters, and classes, matching how the school is actually organized (batches, generations, semesters, programs). This is the biggest project I've worked on, built by a team, and it's a real system that gets used by a real school.

Manage people

Invite users, give them roles, reset passwords, turn accounts on or off.

Manage courses

Set up programs, courses, semesters, batches, and classes.

Flexible permissions

A system that decides exactly what each type of user is allowed to see and do, and can be changed without touching the code.

My role: Frontend. I worked on the pages users actually see and click through — building the screens for managing users, courses, classes, and the roles-and-permissions settings, and making sure they talked correctly to the backend.

what's built so farKey Features

  • Login with email/password or Google, with secure sessions.
  • Invite new users, assign roles, reset passwords, and turn accounts on or off.
  • Manage programs, courses, semesters, batches, and classes.
  • A roles-and-permissions system where each permission can be turned on or off per role, instead of roles being hardcoded in the code.
  • Different dashboards depending on whether you're a Super Admin, Director, Lecturer, or Student.
  • Lecturers can see their classes, students, and courses, and can manage their own classes.
  • Students can see their classes, courses, and lecturers.
  • Director can see all users, batches, courses, and classes, and can manage them.
  • Admins can see logs of all actions taken by users, and can see metrics of how many users are using the system and what they're doing.
