CS128

Online Marketplace for Handmade Products

An online marketplace where people can list and sell handmade products, with an admin panel for managing everything behind the scenes.

by Yongye Uy · CS128 group project

GitHub Repo (backend only)

what it doesOverview

This project is a simple online shop built for handmade products. Sellers can list what they're selling, buyers can place orders, and an admin can manage the whole store from one panel.

Admin panel

A place to manage products, orders, and users without touching the database directly.

Orders

Customers can place orders on products listed in the store.

Login system

Users can create an account and log in to use the marketplace.

My role: Backend Developer. I built the backend using Laravel, including the database structure and the logic behind products, orders, and users, using MySQL to store everything.

built withTech Stack

Laravel PHP MySQL

what's builtKey Features

  • Admin panel to manage products, orders, and users.
  • Product management — adding, editing, and removing products for sale.
  • Order handling for customers buying products.
  • Login system with user accounts.
  • User management for keeping track of who's using the marketplace.
